Telegram Parser Bot

Telegram Parser Bot — это Python-проект для сбора, фильтрации и анализа сообщений из Telegram-каналов.

Проект позволяет добавлять каналы, собирать сообщения за выбранный период, очищать их от мусора, сохранять в базу данных и формировать ИИ-сводки по найденным материалам. Также предусмотрен режим поиска по пользовательскому запросу: бот подбирает релевантные каналы и сообщения, после чего формирует краткий обзор.

Возможности:
  добавление и управление списком Telegram-каналов;
  сбор сообщений из открытых каналов;
  фильтрация рекламы, мусора, коротких и нерелевантных сообщений;
  сохранение сообщений и результатов обработки в базу данных;
  выборка сообщений по ключевым словам и тематике;
  ИИ-разметка каналов по категориям, формату и тематике;
  генерация кратких сводок по собранным сообщениям;
  отчёты о запуске и ошибках;
  защита от чрезмерного чтения Telegram с учётом пауз и FloodWait.
  
Технологии:
  Python
  aiogram
  Telethon
  PostgreSQL
  asyncpg
  APScheduler
  AI API для генерации сводок и разметки
  
Основная идея

Проект нужен для полуавтоматического мониторинга Telegram-каналов. 
Пользователь задаёт тему или выбирает каналы, бот собирает подходящие сообщения, очищает их, отбирает наиболее релевантные и передаёт их ИИ для создания краткого отчёта.

Запуск
Создать виртуальное окружение:
  python -m venv venv
  Активировать окружение:
  venv\Scripts\activate
  
Установить зависимости:
  pip install -r requirements.txt

Создать .env файл с локальными настройками.
